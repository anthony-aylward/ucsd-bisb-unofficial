Check out this useful resource to jump-start your understanding of modern next-generation sequencing technologies and software analysis pipelines.

Brett Copeland posted a collection of useful scripts to the BISB student slack. He wrote:

I've got a collection of scripts that help in viewing files on the command line; I didn't create most of them, but three I can suggest using are the following:

  • sheet.pl (other two are just wrappers to it) (this helps viewing any regularly delimited file, e.g. csv, tsv); use sheet.pl -d | less -S
  • view_vcf (passes whatever other arguments you use onto tabix, so can subset to an indexed file readily)
  • view_bam (passes other arguments to samtools view)

These help a lot in being readily able to read and interpret these files.
